# CookieClicker

Une version stellaire du jeu Cookie Clicker.

## Pincipe du jeu :

Clique sur la fusée pour gagner des points.

En augmentant ton score, tu as la possibilité de débloquer de plus en plus de références geek.

Addiction assurée.
